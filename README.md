## Sobre o Projeto

O **Projeto Amparo** é uma iniciativa social que visa criar uma plataforma digital para apoiar, informar e acolher vítimas de violência doméstica e sexual.


## Tecnologias

- [Next.js](https://nextjs.org/) — Framework React fullstack  
- [React](https://react.dev/) — Biblioteca para UI  
- [TypeScript](https://www.typescriptlang.org/) — Superset tipado de JavaScript  
- [TailwindCSS](https://tailwindcss.com/) — Estilização moderna e responsiva  
- [shadcn/ui](https://ui.shadcn.com/) — Componentes acessíveis e reutilizáveis  
- [lucide-react](https://lucide.dev/) — Ícones modernos  


##  Como rodar o projeto

Clone o repositório e instale as dependências:

```
git clone https://github.com/seuusuario/projeto-amparo.git
cd projeto-amparo
npm install
```


## Funcionalidades

- Página inicial informativa
- Área de recursos e materiais de apoio
- Sistema de autenticação de usuários
- Dashboard administrativo
- Acessibilidade e suporte multilíngue
